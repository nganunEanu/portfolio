Measuring Human Reaction Time with IoT

From Simple Stimulus to Real-Time Insight ⚡

How fast do you react when something suddenly flashes or beeps? That simple question can reveal a lot about your alertness, fatigue, and focus. This project transforms that idea into a measurable system by combining hardware triggers, user input, and real-time data visualization.

🧠 From Stimulus to Insight

The system delivers light and sound stimuli using a buzzer and LED/screen. When triggered, the user must respond as quickly as possible by pressing a button.

The time difference between stimulus and response is recorded as reaction time, which can be used to:

  • Evaluate fatigue levels
  • Monitor alertness
  • Analyze focus over time

All collected data is processed and visualized using Node-RED, enabling real-time monitoring through an intuitive dashboard.

🔌 How the System Works

  • Trigger System → Buzzer and LED/screen provide random stimuli
  • User Input → Push button captures response timing
  • Processing Unit → Microcontroller calculates reaction time
  • Visualization → Data sent to Node-RED dashboard in real-time

This creates a closed-loop system where human response is instantly measured and displayed.
