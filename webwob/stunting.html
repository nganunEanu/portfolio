Smart Stunting Detection with 3D Scanning & AI

Transforming Health Screening with Computer Vision and Machine Learning 🧍‍♂️

Early detection of stunting is crucial for effective health intervention, but traditional measurement methods can be time-consuming and inconsistent. This project introduces a smart, automated system that combines 3D body scanning, machine learning, and an intuitive interface to make assessment faster, more accurate, and easier to use.

📡 From Body Scan to Prediction

The system operates as a complete data pipeline:

  • Rotating platform captures the subject from multiple angles
  • Depth sensor collects 3D body measurements
  • Data is processed and fed into a machine learning model
  • Prediction is generated in real-time

The system uses Microsoft Kinect to capture skeletal tracking and depth data, enabling accurate, non-contact measurement.

🧠 Machine Learning for Health Assessment

The prediction engine is developed using Keras, trained on anthropometric features extracted from the scanning process.

  • Automatic extraction of body dimensions
  • Conversion into structured data
  • Prediction of stunting condition

This approach improves speed, consistency, and reduces dependency on operator skill.

🖥️ Human–Machine Interface (HMI)

A graphical interface built with Tkinter serves as the central control hub for the system.

  • Start/stop scanning process
  • Control rotating platform
  • Display measurement results
  • Show prediction outcomes

The interface is designed to be intuitive, allowing use by non-technical operators.
