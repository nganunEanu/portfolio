Cashless Battery Exchange System with NFC & IoT

Building a Smart Energy Transaction System for My Undergraduate Thesis 🔋

As energy systems evolve, the need for smarter, faster, and more user-friendly infrastructure becomes increasingly important. This project—developed as my undergraduate final thesis—introduces a battery exchange system that eliminates cash payments by integrating NFC-based digital transactions.

The goal is simple: create a seamless, efficient, and scalable solution for battery exchange operations in modern energy ecosystems.

⚙️ How the System Works

The system manages the full workflow of a battery exchange transaction:

  • User taps an NFC card
  • System reads user data and balance
  • Data is verified via backend server
  • Battery exchange process is executed
  • Transaction is recorded in real-time

At the core of the system is the ESP32, which manages communication, control logic, and system coordination.

🔌 Key Technologies Used

  • NFC Reader → MFRC522 for reading user cards
  • HMI Display → Nextion display for user interaction
  • Communication Bus → MCP2515 (CAN bus) for inter-module communication
  • Backend Server → PHP-based system for transaction and balance management

The system combines WiFi for cloud communication and CAN bus for reliable internal data exchange between modules.

🛠️ My Role in the Project

I handled both hardware integration and software development, making this a full-stack embedded system project.

💻 ESP32 System Programming

I developed the main control logic, including NFC reading, system state handling, and coordination between all modules.

📡 NFC Transaction System

Using MFRC522, I implemented card detection, UID reading, and backend validation to enable secure and seamless transactions.

🔗 CAN Bus Communication

I configured MCP2515 for reliable real-time communication between subsystems, including message structuring and parsing.

🖥️ HMI Design (Nextion)

I designed the user interface to provide clear instructions, smooth navigation, and real-time feedback during transactions.

🌐 Backend Integration

I implemented WiFi communication with a PHP backend to validate balances, update transactions, and maintain data consistency.

🚀 Why This Project Matters

  • ✅ Eliminates cash handling in battery exchange systems
  • ✅ Enables fast and user-friendly transactions
  • ✅ Integrates IoT with energy infrastructure
  • ✅ Provides real-time monitoring and data management

🌟 Final Thoughts

As my undergraduate final thesis, this project represents the integration of multiple disciplines—from embedded systems and communication protocols to user interface design and backend development.

It’s not just about swapping batteries—it’s about building a smart, connected, and user-centric energy solution ready for real-world implementation.
